DATA CLEANING AND VISUALIZATION PROJECT

Project Objective:
The objective of this project is to clean raw data and visualize useful insights using Python libraries.

Tools and Libraries Used:

* Python
* Pandas
* Matplotlib
* Seaborn
* VS Code

Tasks Performed:

1. Loaded dataset using Pandas
2. Identified missing values
3. Filled missing values using mean and default values
4. Removed duplicate records
5. Detected outliers using boxplot
6. Created bar chart visualization
7. Saved cleaned dataset

Files Included:

* students.csv (raw dataset)
* cleaned_students.csv (cleaned dataset)
* project.py (Python code)
* README.txt (project summary)

Outcome:
Learned data preprocessing, cleaning, visualization, and storytelling with data using Python.
